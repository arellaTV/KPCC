# KPCC
> Web application that allows a user to search for KPCC articles.

## Table of Contents
1. [Usage](#usage)
1. [Stack](#stack)
1. [Development](#development)
    1. [Installing Dependencies](#installing-dependencies)

## Usage
From within the root directory
```sh
$ npm start
```

## Stack
- Node 6.3.1
- Express
- React
- Mocha/Chai/Enzyme

## Development
### Installing Dependencies
From within the root directory:
```sh
$ npm install
```

## Testing
From within the root directory:
```sh
$ npm test
```
